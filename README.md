# Staysafe-project
## Description
A Quarkus application designed to help people in war zones find shelters. Each microservice is in a separate repository, making maintenance straightforward.
## Auth Service
This service is used to register, login and generate token for users

   https://github.com/MedKramti/staysafe-auth-server

## Shelters Service
This service is used to manage shelters

   https://github.com/MedKramti/staysafe-shelters-microservice

## Frontend
This is the frontend of the application

   https://github.com/MedKramti/staysafe-frontend

## Docker Compose
To easy deploy the application I created a docker compose file

   https://github.com/MedKramti/staysafe-docker-compose
